+ 原则
    + 多使用内存、缓存或其他方法
    + 减少CPU计算、减少网络

+ 入手
    + 加载页面和静态资源
        + 静态资源的压缩合并
        + 静态资源缓存
        + 使用CDN让资源加载更快（不同区域的网络优化）
        + 使用SSR后端渲染，数据直接输出到HTML
    + 页面渲染
        + CSS 放前面，JS 放后面
        + 懒加载（图片懒加载）
        + 减少DOM查询，对DOM查询做缓存
        + 减少DOM操作，多个操作尽量合并在一起执行
        + 事件节流
        + 尽早执行操作（DOMContextLoaded）
